# Business Booking Bot

Telegram бот для записи клиентов на услуги салона красоты.

## Функции
- Запись на услугу с выбором даты и времени
- Просмотр своих записей
- Отмена записи
- Уведомление администратору о новой записи

## Стек
Python, python-telegram-bot, ConversationHandler

## Запуск
1. Создай .env файл с BOT_TOKEN и ADMIN_ID
2. pip install python-telegram-bot python-dotenv
3. python bot.py
